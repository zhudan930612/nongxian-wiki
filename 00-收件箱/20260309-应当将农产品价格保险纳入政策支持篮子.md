自从 2007 年中央和各级财政开始为农业保险补贴保险费以来，中国农业保险规模实现了年均两位数的快速增长。到 2024 年末，农业保险保险费的规模达到了 1521 亿元，成为全球之首。

但是，在财政政策上，中央至今只支持 17 大类 20 多个标的的自然灾害风险保障产品，并不支持农产品生产的价格风险保险。虽然目前各地试验的 “收入保险”有价格保险的元素。但明确的政策是不支持价格保险，尽管中央各种重要文件一再鼓励试验和发展“保险 + 期货 ”。

本文主要讨论为什么需要将价格风险纳入中央支持的政策性农业保险的范围之内。

**一、价格风险是影响农户收入的重要因素之一  
**

（一）中国农产品价格的市场化改革

自上世纪 80 年代我国启动市场化改革进程以来，我国农产品价格逐步放开，其形成机制持续向市场主导转型。在 2015 年之前，国家仅保留水稻、小麦、玉米、棉花和大豆五种大宗农产品的价格保护政策，每年都对这些农产品实行保护价收购，以保证在市场价格剧烈下跌的时候这些作物生产者的收入稳定。 2015 年到 2017 年，国家先后对棉花、大豆和玉米实行市场化定价机制改革，到目前，仅仅保留了水稻和小麦两种产品继续实行价格保护政策。

这场市场化改革既是我国经济体制改革的重要部分，也是我国对世界贸易组织（ WTO ）的承诺。在 WTO 的《农业协定》里，我国承诺逐步放开所有农产品的市场和对农产品的价格补贴。我们基本上兑现了承诺，目前除了水稻和小麦之外的所有农产品都实行了完全的市场化，不再有政府的价格补贴。

（二）农产品价格的波动对农户收入影响很大

对于农业生产者而言，价格风险是构成其收入不确定性的核心因素之一。自古以来就有 “谷贱伤农”的精辟论断，深刻揭示了市场价格下行对农业再生产与农民收益的负面冲击。特别是这些年来，因为市场价格下跌对农业生产和农户收入造成的影响不小。以玉米为例， 2016 年临储政策调整后，价格波动风险尤为凸显。根据中国玉米网等专业机构的数据， 2017 年中国玉米主产区（如黑龙江）玉米收购价大约在 1500-1700 元 / 吨区间内宽幅震荡。以前一年政府临时收储价为对比基准，玉米价格下跌幅度约为 0.15-0.25 元 / 斤。按平均跌幅 0.2 元 / 斤、 2017 年全国玉米产量 2.16 亿吨、商品化率 75% 估算，损失约为 648 亿元。尽管后续年份玉米市场价格有所回升，但其波动性始终是高悬于我国农户收入之上的 “达摩克利斯之剑”。其他的农产品亦面临类似的风险挑战，特别是水果、蔬菜、天然橡胶等产品，市场价格大幅波动给生产者带来的损失往往是很大的。虽然，从经济学理论而言，农产品价格波动是市场对市场供给与需求调整的“看不见的手”，这是有利于资源合理配置的重要信号和手段。但是价格较大的波动，特别是大幅度下跌，不仅直接侵蚀农户的宝贵 经营利润，更会严重挫伤其扩大再生产的积极性，甚至会危及农业产业的可持续性与国家粮食安全。因此，农产品价格风险保障 是必要的，特别是构建有效的、精准的、适度的、可持续的农产品价格风险保障机制，这无论是从农业安全视角还是乡村振兴的视角，都是需要解决的。

**二、“农业保险+期货”开启了农产品价格保险的先河  
**

（一）农产品价格风险是否可保

价格风险虽然对农业生产的可持续发展和农民收入的稳定影响很大，在某种情况下甚至比自然风险带来的农业损失还要严重。 “增产不增收”一直是困惑农业和农民的重要问题。但是价格风险在保险界被认为是不可保风险主要是因为这种价格风险具有系统性的特征，无法通过大数法则在众多的投保人和被保险人中间进行有效分散，也就是说无法在空间维度进行有效分散。尽管因为价格变动具有周期性，理论上在一定程度上可以在时间维度上进行分散与平滑。但是，这对保险人来说是一个非常大的挑战。所以，保险人和再保险人都拒绝承保和接受分保价格风险责任。

自 1848 年芝加哥期货交易所开创具有现代意义的商品期货交易以来，全球期货市场蓬勃发展，并从商品期货发展到金融期货。期货市场的发展就是适应了市场对价格风险分散的需求。因为期货市场不仅通过公开交易实现价格发现，更能通过风险转移为生产者提供套期保值工具，从而在一定程度上对冲现货市场的价格波动，缓解价格变动对商品生产者的冲击。

**（二） “农业保险** **+** **期货 ”是中国对世界的贡献**

2015 年大连商品交易所与中国人保财险合作开发出我国第一款 “保险 + 期货 ”农产品期货价格保险，该创新模式于次年被写入中央“一号文件”，并在此后连续八年的中央“一号文件”中获得肯定和支持。表明中央对于在农产品定价市场化过程中，农产品价格波动造成的损失高度关切，期望能通过保险业与期货业的合作，帮助日益现代化的中国农业解决价格风险保障问题，帮助农户稳定农业收入，迅速脱贫和加快推进乡村振兴。

在 2015 年之后的十年， “保险 + 期货 ”迅速在全国各地开始试验，创新出几十种新产品，特别是在“价格保险 + 期货 ”的基础上，进一步开发出“收入保险 + 期货 ”，从单纯保价格进入承保农业收入。不仅在“保险 + 期货 ”的定价方面做出了有益探索，也在“保险 + 期货 ”操作方面有了不断发展和完善。这是在价格保险方面做出的贡献。当然，美国从 1996 年就开始试验农产品收入保险，尽管也参考了期货市场的价格，但是与中国的 “收入保险 + 期货 ”的思路还是有区别的。美国的期货行业并没有介入收入保险。

中国的 “保险 + 期货 ”的试验经验受到联合国粮食计划署（ WFP ）的关注，在中国多地的调研之后，亲自组织资金在吉林进行试点，并准备向其他发展中国家推广。

**（三） “农业保险** **+** **期货 ”的试验发展成就巨大**

十年来，在三大商品期货交易所、几十家期货公司和保险公司的共同努力下， “保险 + 期货 ”试验逐步扩大，取得了丰硕的成果。据三大商品期货交易所的报告显示：

自 2017 年 “保险 + 期货 ”项目启动以来，上海期货交易所联合保险公司、期货业以及地方政府等多方力量，连续八年在海南、云南等地开展天然橡胶“保险 + 期货 ”项目，累计补贴保费约 8.2 亿元，保障了约 103 万吨天然橡胶。此举不仅为橡胶种植户的收入提供了实质性保障，也为当地橡胶产业持续发展提供了坚实保障；同时为国家战略性资源供应提供了有力支持，并且促进了国家相关政策的有效落地实施。

十年来，大连商品交易所持续优化和完善 “保险 + 期货 ”模式，“保险 + 期货 ”成为广大农户稳收增收的“保护伞”。截至 2024 年底，大商所在 31 个省（自治区、直辖市）累计支持开展了 1338 个 “保险 + 期货 ”项目，保障玉米、大豆、鸡蛋、生猪等品种现货 2015 万吨，共实现赔付超 20 亿元，惠及农户 183 万户次。

2025 年是郑州商品交易所开展 “保险 + 期货 ”的第十年，这些年郑商所结合试点成效与期货品种上市进展，持续探索、研究与优化，不断增加规模、扩大区域，累计支持开展项目 250 个，覆盖全国 16 个省级行政区域，涉及苹果、红枣、花生、白糖、棉花与油菜等 6 个品种。

当然，目前 “保险 + 期货 ”不仅得到了三大期货交易所的支持，也得到地方政府的支持，众多期货公司也贡献了宝贵的智慧和辛劳。

**三、“农业保险+期货”试验和发展面临的困难  
**

“农业保险 + 期货 ”通过中国式的金融工程创新，在风控技术上实现了农产品价格风险的可保性，初步打开了一条为农产品价格提供风险保障的道路。将传统保险理论因“不可分散”而“不可保”的系统性价格风险，创造性转化为在金融市场上可交易、可对冲，进而可管理的市场风险。

但是， “农业保险 + 期货 ”的试验要得到进一步推进并最终成为我国农业风险保障制度的组成部分，还存在不少实际问题。

（一）价格保险尚未纳入农业保险的政策 “篮子”

目前的政策性农业保险的保险责任，在政策上只承保自然风险，不承保价格风险。有关部门囿于传统认知，也一直认为价格风险不可保， “保险 + 期货 ”不属于政策支持的范围，因为期货、期权是市场化产品。而如果不将“保险 + 期货 ”纳入政策性农业保险的轨道，不能将政府补贴制度化，农户独立承 担高昂的 “保险费”或者期权交易费（保证金），“农业保险 + 期货 ”就不可能做下去。

目前的实际情况是，因为只靠三大商品期货交易所和地方政府的这种 “项目制”而非“制度性”的政策资金支持方式，很可能使得“农业保险 + 期货 ”的长期发展前景面临极大的不确定性，难以实现从试点实验创新到普惠性政策的根本性历史跨越。

**（二） “农业保险** **+** **期货 ”“保险费”的困惑**

根据目前保险业会计准则， “保险 + 期货 ”不被定义为“保险”，特别是“价格保险 + 期货 ”的“保险费”不算保险费收入，而保险公司本来也只是收取佣金（所谓“通道费”）。当然，现行“价格保险 + 期货 ”产品对保险公司来讲并没有承担风险责任。虽然从传统保险原理上说，也没错。

还有，按照新的会计准则， “保险 + 期货 ”做期权对冲的收入（如果有的话），也不能算做保险收入，只能算作“营业外收入”。

但是这些规则下的会计处理方式，尽管在技术层面上、传统的保险原理上符合 “风险转移”不充分的特征，却可能产生与国情和金融工作本质相悖的负面效应。其核心问题在于，这种规则模糊了保险公司的业务属性，扭曲了其业绩核算，从而在根本上削弱了保险公司持续投入资源进行产品创新与市场拓展的内在激励。简言之，当前准则在审慎确认风险的同时，也在客观上抑制了保险主体继续试验和推广“保险 + 期货 ”这一重大金融创新的积极性。

**（三） “农业保险** **+** **期货 ”仅靠市场难以发展**

农业保险因为市场失灵不可能发展商业保险，或者说农业保险的商业化市场化经营不可能有有效需求，即使做起来也不可持续。也正是因为农业保险本身具有准公共物品属性，这才导致了政府的介入，通过政府的保险费管理费补贴，形成一个特殊的政策性农业保险市场。

而期货也很难走进农户，特别是我国的小农户。除了专业和技术知识局限外，农户一般也难以承受期货期权的交易成本。 “农业保险 + 期货 ”的诞生，就是想借“政策性农业保险”之光，使期货期权能为农户的价格风险分散的需求服务。就是说，作为平滑农产品价格风险工具的期货期权工具，也需要政策支持，从而创新出专门为农户服务的“政策性期货”产品，获得政策支持，特别是财政支持。这就是“农业保险 + 期货 ”的政策本质。也是“农业保险 + 期货 ”的光明前景所在。

据笔者的有限考查，即使在发达国家，农户自己进入期货市场进行交易的也很少，有的农场主也只是通过期货中介人来替自己进行期货期权交易。因此，在中国，只有实现了农业规模化、集约化、现代化之后，才有可能让那时候的农民直接或通过期货公司进入完全商业化的期货市场。

**（四） “农业保险** **+** **期货 ”操作中的失范和违规违法问题**

由于监管规则缺乏和监管力度不够，在 “农业保险 + 期货 ”操作中存在失范和违规违法问题。

1.期货公司信用资质不容忽视

全国期货公司中高评级（ AA 及以上）主体占比低，整体资本实力和抗风险能力偏弱。在面临巨大市场波动时，存在无法履行场外期权赔付义务的信用违约风险。

2.对冲 “不作为”

部分期货公司在收取期权费后，如未进行足额、有效的对冲，甚至没有对冲，这实质上是一种变相的违约和利益输送，将风险留给了保险体系和财政资金。

3.利用模式漏洞进行投机套利与道德风险

部分参与者可能并非服务 “三农”，而是钻研产品细节，利用期现价差、产品设计漏洞进行套利，使风险管理工具异化为投机工具。虚假投保与理赔，存在虚构标的、夸大损失以套取财政补贴和理赔款的道德风险。

4.考核驱动下的规模虚增与数据造假

为满足业绩考核与监管评级要求，部分机构可能通过拆分保单、虚报承保规模等方式 “刷数据”，导致行业统计失真，并可能基于虚增规模放大市场风险。

5.期货公司的非正当竞争

为抢占市场，部分机构进行低于成本的恶意报价，扰乱市场秩序，长期看必然导致服务质量下降或风险积累。

6.合规性存在缺失

协议赔付，保底赔付，甚至制造假标的，假合同套取财政补贴，还有保险公司的 “农业保险 + 期货 ”合同，采用超短期期限，没有完全匹配生产周期等。

这些问题其实在一定程度上损坏了 “保险 + 期货 ”的形象，违规操作也就失去了发展这种创新产品的意义。

**（五）对 “农业保险** **+** **期货 ”的监管规则不完善**

十年实践表明， “农业保险 + 期货 ”的试验和发展离不开规则，也离不开监管。因为这十年一直处于试验阶段，没有完整的监管规则，呈现一种“政出多门、分割管理”的状态。证监部门、保监部门、财政部门、税务部门等，都是从各自业务角度做出了一些规定，但没有统一政策思路，也没有统一监管思路，导致“农业保险 + 期货 ”的试验并不是那么顺畅。

比如，现在的资金支持方主要是三大商品期货交易所和地方政府，这是非常关键的问题，却没有任何规则。对不同的 “保险 + 期货 ”产品，要不要补贴，补贴多少，谁来补贴，带有较大的随意性，而非基于明确的法律或政策依据。

再比如， “收入保险 + 期货 ”中为了降低价格风险，要在期货 市场进行对冲，但是这部分 “保证金”的支出，按照新会计准则，是要计入保险费用成本，但是对冲的收入，却要计入“营业外收入”，这对“单独核算”的农业保险还是有影响的。不利于农业保险的发展。至少不能准确反映农业保险的经营成果。

**四、推进“保险+期货”的试验和发展需要解决的主要问题  
**

**（一）加强对农产品价格保险和 “农业保险** **+** **期货 ”的基础理论研究**

做农产品价格保险，离不开期货市场， “保险 + 期货 ”是比较好的选择。但是，对于农产品价格保险和“农业保险 + 期货 ”，我们的研究还不够深入，我读过的有限文献，给我一种印象，那就是，学界和业界对于具体的产品及其效果的研究比较感兴趣，而对农产品价格保险，“农业保险 + 期货 ”的基础研究有些欠缺，对于为什么要开发农产品价格保险，农产品价格波动对农户收入影响几何，农产品价格保险的可行性如何，为什么需要财政支持农产品价格保险和“农业保险 + 期货 ”。如果需要政府补贴，补什么，补给谁，怎么补，补多少，财政的支持力度等问题，这些问题没有研究清楚，不能提供可行性方案，很难让决策部门做出选择。

农业保险为什么需要政府补贴的问题，在理论界做了很多研究，得出了明确的结论。业界和学界也对于农业保险需要补多少，补哪些产品，需要多少财政预算等问题，都做了细致研究和测算，拿出了可行的方案。这才有 2007 年的财政出手。对于 “农业保险 + 期货 ”如果没有充分的论证，拿不出充分的依据，进入政策支持的“篮子”是很难的。

（二）将价格风险纳入政策性农业保险制度之中

目前的政策性农业保险，保险责任只是自然灾害、地质灾害和意外事故造成的损失，不包括价格风险。如前所述，价格风险也确实在很大程度上影响到农户农业收入的稳定性，这对我们加快推进乡村振兴，提高农户的农业收入，缩小城乡收入差距，有重要意义。无论是直接承保价格风险还是通过 “收入保险”方式承保价格风险，都是需要进一步明确的。对于政策性农业保险与期货的合作，需要政府继续支持，并在可行性研究基础上，将“农业保险 + 期货 ”纳入政策性农业保险制度之中，并给予合理的保险费补贴，这是解决农产品价格风险问题所不能回避的。

**（三）科学合理给 “农业保险** **+** **期货 ”定价**

做政策性保险有农业保险的精算模型和相应的精算规则， “农业保险 + 期货 ”的定价也要有理论基础和相应的模型或者规则，不能只靠保险公司和期货公司双方，保险公司、期货公司和政府之间的协议定价，随意性很大，透明度低，准确性太差，不那么科学，既难以保证公平，无法准确反映风险成本，也没有办法评估其合理性和适当性，还会产生其他问题。

所以也要加强 “农业保险 + 期货 ”定价理论和实务规范的 研究，通过监管部门制定一套科学、统一、透明的 “农业保险 + 期货 ”的定价规则，才有利于“农业保险 + 期货 ”作为政策性业务的顺利发展。

**（四）在 “农业保险** **+** **期货 ”的实现模式上做好文章**

目前大部分 “农业保险 + 期货 ”产品都是同一类操作模式，即对于“农产品价格保险 + 期货 ”是由保险公司出具“价格保险单”，收取事先约定好的“保险费”，然后将“保险费”扣除“通道费”（佣金）后全部交给期货公司，由期货公司进行场外期权交易，如果约定期内期货价格（一个月平均）低于保险合同约定的价格，将由保险公司赔偿给投保农户合同约定的几个与期货市场价格的价格差。“农业收入保险 + 期货 ”的操作方式主要是，保险公司与期货公司分别承担产量和价格风险，协商分配保险费（比如， 50% ： 50% ，或者 60% ： 40% ），对于承保农户的收入损失，如果是因为产量造成，就全部由保险公司承担，如果是由价格下降造成，就全部由期货公司赔偿。如果收入有两个因素共同造成，就共同承担损失补偿。这种操作方式在现行监管制度下，有政策障碍。

2023 年 11 月在勐腊召开的 “保险 + 期货 ”研讨会上，笔者曾提出另一种“风险分层”的实现模式，设计出“先确立保险关系，后进行风险对冲”的理论路径。在签发“农业收入保险”保单时，由保险公司作为唯一的风险承担主体，面向农户承接全部收入风险。此后，保险公司再将其中难以独立承保的系统性价格风险，通过两种市场化手段进行转移：一是可以委托期货公司在期货市场进行操作；二是可以由保险公司自身的资金运用部门直接在期货市场进行对冲操作。这种“先保险、后期货”的流程设计，将保险合约与衍生品交易在时间和主体上清晰分离，可能更符合当前金融业分业监管的法规框架，能有效规避混业经营的政策障碍。

我们认为，应鼓励此类模式创新，需要大家通过集思广益，设计出既能有效管理风险，又符合监管规定、操作便捷的实现模式与路径。

**（五）加强对 “农业保险** **+** **期货 ”的统一监管**

由于对农产品价格保险的固有观点，对于无论是价格保险还是 “农业保险 + 期货 ”一直处于试点和观察阶段，这种常态化停滞于“政出多门”的分散监管状态，是制约该模式可持续健康发展的最大体制性障碍。金融监管部门和财政部门等基于自身职责出台的零散政策，因为缺乏必要的顶层设计与协同，从而使得发展不顺畅。如果要继续推进“农业保险 + 期货 ”的试验，甚至要推而广之，当务之急，就需要相关部门在统一认识的基础上共同制定统一的业务指引、会计处理与统计标准，对“农业保险 + 期货 ”进行联合的功能监管与穿透式监管，而不是各自为政。这也许是肯定或者否定“农业保险 + 期货 ”最优的路径。
