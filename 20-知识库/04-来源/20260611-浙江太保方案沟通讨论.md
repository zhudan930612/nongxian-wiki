# 浙江太保方案沟通讨论会议纪要

> 对浙江太保项目建设方案进行逐页评审，重点纠正方案标题、定位、文字表述及报价策略，明确项目核心为"AI智能风控平台+协保员队伍建设"两大模块。

## 核心观点

1. **方案标题修正**：统一为"太平洋财产保险浙江分公司农保云AI智能风控平台解决方案"，去除空泛表述。
2. **项目定位纠偏**：从"汇报方案"改为"建设方案"，删除监管要求和口号式表述，聚焦企业经营自身需求。
3. **核心内容拆分为两大块**：一是AI智能风控平台（AI中台+道德风险防控智能体），二是协保员队伍建设。
4. **报价策略设计**：总费用130万（基础110万+劳务20万），预留讨价还价空间，控制在150万以内。
5. **文字严谨性要求**：杜绝口语化、自创名称，使用行业标准术语。

## 关键内容

### 方案背景重新定位
- 浙江农险连续多年亏损，赔付率高，道德风险防控是企业扭亏为盈的经营需求，而非监管要求
- 删除"监管标准高""覆盖面广"等不准确表述
- 逻辑链条：行业亏损现状 -> 建设系统降本增效 -> 依托协保员体系业务协同

### 项目目标与内容
- 项目目标三个：AI智能风控平台建设、道德风险防控智能体开发、协保员管理工具建设
- AI智能风控平台=AI智能体中台+道德风险防控智能体，未来可扩展其他智能体
- 协保员队伍建设：信息化工具+人员遴选增补（从灾害信息员中遴选）

### 方案结构要求
- 简化结构：项目背景 -> 项目目标 -> 建设内容 -> 实施阶段 -> 双方分工 -> 项目费用
- 删除"预期成效""保障措施""合作总结"等冗余章节
- 文字表述必须专业准确，杜绝口语化、自创名称

### 项目报价设计
- 总费用130万（基础信息技术服务110万+新增协保员劳务费20万）
- 预留议价空间，控制在不低于98万、整体150万以内
- 劳务费实报实销可谈
