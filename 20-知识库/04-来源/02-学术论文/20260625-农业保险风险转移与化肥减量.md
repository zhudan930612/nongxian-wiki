# 农业保险风险转移与化肥减量

> 基于 2011—2022 年我国粮食主产省县级面板数据，以完全成本保险试点为准自然实验，考察农业风险管理对单位面积化肥用量的影响。

## 核心观点

1. **研究问题**：发展中国家农户风险规避倾向普遍偏高，可能通过过量施用化肥来规避产量与收益不确定性，农业保险能否缓解这一现象？
2. **研究方法**：双重差分法（DID），以完全成本保险试点为外生政策冲击。
3. **主要结论**：
   - 完全成本保险试点具有显著的化肥减量效应
   - 机制：完全成本保险通过降低农业收入波动促进了化肥减量
   - 异质性：试点更有助于降低自然灾害高发地区和农地经营规模更高地区的单位面积化肥用量
4. **理论解释**：根据前景理论，农户对损失更为敏感。高保障水平的完全成本保险提高预期收入、降低收入波动，使农户探索更接近利润最大化目标的施肥比例。
5. **政策建议**：
   - 提高农业保险法制化水平，推进风险管理正式制度建设
   - 实施差异化的费率和保费补贴政策
   - 加大对自然灾害高发和农地经营规模更高地区的政策支持力度
   - 规范理赔制度，强化农业保险的稳收效果
