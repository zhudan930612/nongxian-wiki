# 国家减灾中心-张云霞-灾害信息员队伍沟通

> 与张云霞主任沟通灾害信息员队伍转化为农险协保员的可行性、认证体系设计、AI中台产品架构及与国家减灾中心的合作边界。

## 核心观点

1. **队伍定位**：灾害信息员队伍属于华宇科技自建平台，不是帮某一家保险公司串联业务，而是面向全行业提供第三方服务能力。
2. **认证体系分层**：初级灾害信息员（基础报灾职能）→ 中级加入协保技能 → 高级加入理赔技能，形成分级认证体系。
3. **数据不出库原则**：国家减灾中心数据不能出库，保险公司数据也不能出库，通过地理位置信息比对，仅传递异常结果。
4. **合作边界清晰**：国家减灾中心只做普惠性培训、认证平台建设，任务下发、商业运营由华宇科技自建平台承担。

## 关键内容

### 协保员队伍管理
- **身份属性**：灾害信息员多为村两委委员、社区工作人员，非保险公司劳务员工，也非减灾中心行政下属。
- **招募逻辑**：自愿报名，非强制要求，有意愿者加入，通过培训考核后纳入平台管理。
- **费用补贴**：基础费用约2000元/人/年，由保险公司支付；额外任务按件计费。
- **覆盖模式**：不是按人头全覆盖，而是按村覆盖（2000元/村），一个信息员可覆盖多个村。

### 认证体系设计
- **初级**：基础灾害信息员，面向现有120万人，免费培训。
- **中级**：加入农险协保技能，可作为协保员角色，需付费认证。
- **高级**：加入理赔、展业等保险专业技能，面向更专业场景。
- **价值**：认证后从"职能"变为"职业"，具备专业能力和职业资格。

### AI中台与产品架构
- **三层架构**：
  - 业务应用与数据层（保险公司自建或华宇代建）
  - AI中台（华宇建设，核心智能体解决道德风险防控）
  - 灾害信息员协同网络（华宇建设，地面任务执行）
- **核心场景**：长势监测→异常提取→与保单比对→形成问题清单→下发信息员现场核查→证据回传→业务闭环。
- **数据融合**：国家减灾中心遥感数据 + 华宇AI中台规则 + 保险公司保单数据，形成道德风险防控解决方案。

### 与国家减灾中心合作模式
- **共建产品**：联合开发遥感数据应用产品，知识产权双方共有，面向全国保险行业发布。
- **数据服务**：国家减灾中心提供卫星数据、算法、无人机资源；华宇负责产品化、市场化。
- **培训认证平台**：共建灾害信息员培训认证平台，分级分类，全国推广。
- **收益分成**：华宇向保险公司收费，按比例返还给国家减灾中心。

### 关键约束
- **数据安全**：国家减灾中心原始数据不出库，仅提供结果数据；保险公司保单数据也不出库。
- **平台归属**：协保员任务平台由华宇自建，非国家减灾中心官方平台，避免行政风险。
- **部门协同**：理想状态是农业农村部+国家减灾中心跨部门合作，将协保员与灾害信息员队伍合并管理。
